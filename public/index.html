Toy Project Documentation

User Routes

METHOD URL BODY DESCRIPTION
POST (register) http://localhost:8000/api/v1/users/register { "email": "test@example.com", "password": "testpassword", "firstName": "John", "lastName": "Doe" } Register a new user.
POST (login) http://localhost:8000/api/v1/users/login { "email": "test@example.com", "password": "testpassword" } Login and receive an authentication token.
GET http://localhost:8000/api/v1/users/ none Get a list of all users.
GET http://localhost:8000/api/v1/users/{user_id} none Get user details by ID. Replace {user_id} with the actual user ID.
PATCH http://localhost:8000/api/v1/users/{user_id} { "firstName": "UpdatedFirstName", "lastName": "UpdatedLastName" } Update user details by ID. Replace {user_id} with the actual user ID.
DELETE http://localhost:8000/api/v1/users/{user_id} none Delete user by ID. Replace {user_id} with the actual user ID.

Toys Routes

METHOD URL BODY DESCRIPTION
POST http://localhost:8000/api/v1/toys { "name": "ToyName", "description": "ToyDescription", "category": "ToyCategory", "price": 19.99, "img_url": "https://example.com/toy_image.jpg" } Add a new toy to the database. Requires authentication token.
GET http://localhost:8000/api/v1/toys/ none Get a list of all toys.
GET http://localhost:8000/api/v1/toys/{toy_id} none Get toy details by ID. Replace {toy_id} with the actual toy ID.
PATCH http://localhost:8000/api/v1/toys/{toy_id} { "name": "UpdatedToyName", "description": "UpdatedToyDescription" } Update toy details by ID. Replace {toy_id} with the actual toy ID. Requires authentication token.
DELETE http://localhost:8000/api/v1/toys/{toy_id} none Delete toy by ID. Requires authentication token.
